# Practica Final

Este repositorio contiene el proyecto desarrollado como **práctica final** de la asignatura, integrando conceptos y técnicas vistos a lo largo del curso.

## Descripción

El proyecto está implementado en **C#** y reúne distintas funcionalidades que ponen en práctica los conocimientos adquiridos durante la asignatura. Se estructura de forma que permite compilar y ejecutar los módulos principales desde un entorno de desarrollo compatible.

## Tecnologías utilizadas

- C#
- .NET
- Entorno de desarrollo (por ejemplo, Visual Studio)
